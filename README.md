# camera101

<img width="635" height="466" alt="image" src="https://github.com/user-attachments/assets/fa294ee0-e446-4b75-a792-2afee41f1aac" />

camera101 is a substitute for the Windows Camera app (Win10/11) and a useful tool for users of older Windows versions
who don't have a built-in camera app / have to deal with a poor propietary camera app.

camera101 does not have any complex features. You can change the camera source (switch cameras), take pictures and view them.
