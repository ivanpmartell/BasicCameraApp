# Basic Camera App

<img width="635" height="466" alt="image" src="https://github.com/user-attachments/assets/fa294ee0-e446-4b75-a792-2afee41f1aac" />

Forked from camera101. It runs on Windows XP to 11. It's a substitute for the Windows Camera app (Win10/11) to avoid the bloated propietary app, and a useful program for users of older Windows versions that don't have a built-in camera app.

camera101 is:

*  lightweight - uses less than 50mb of ram with a camera stream active
*  native - C# WinForms, no UI toolkits/libraries. Uses DirectShow.
*  aesthetically-pleasing - clean, tasteful, glossy theme
*  dark-mode - despite running on Windows XP to 11, it is not going to burn your eyes
*  FOSS - source code is in the repo
*  not a video-taking or screenshotting app - nobody does that better than OBS and ShareX

**Captures are saved to Pictures\camera101**

DirectShow .NET library obtained from https://sourceforge.net/projects/directshownet/files/DirectShowNET/v2.1/
